# Todo List

Todo List is a simple Express and EJS based task manager that supports creating, editing, and deleting tasks in a clean local web interface.

## Tech Stack

- Node.js
- Express
- EJS
- method-override
- UUID

## Features

- Add tasks
- Edit existing tasks
- Delete tasks
- In-memory task handling for quick local use

## Run Locally

```bash
npm install
node route.js
```

Then open `http://localhost:8080`.

## Note

This project is a lightweight CRUD practice app built for learning Express basics.
